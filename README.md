# Codex — EPUB Reader

A clean, minimal EPUB reader PWA with library management, bookmarks, reading progress, and dark mode.

## Deploy to Railway

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/new?repo=https://github.com/eeruc/codex-epub-reader&envs=APP_PASSWORD,PORT&APP_PASSWORDDesc=Password+to+access+the+app&APP_PASSWORDDefault=Zaibatsu&PORTDefault=5000)

Click the button above, then:
1. Sign up / log in to Railway (free, just needs a GitHub account)
2. It will pre-fill the environment variables — just click Deploy
3. Once deployed, go to Settings → Networking → Generate Domain to get your public URL

## Features

- **Library** — Upload EPUBs, search by title/author, drag-and-drop
- **Reader** — Paginated EPUB rendering with tap/swipe/keyboard navigation
- **Table of Contents** — Slide-out chapter navigation
- **Bookmarks** — Save/remove at any page
- **Reading Progress** — Percentage bar, auto-resume where you left off
- **Dark Mode** — Full theme toggle for UI and book content
- **Font Size** — Adjustable 60%–200%
- **PWA** — Installable on iOS and Android home screens
- **Password Protected** — Only accessible with the shared password
